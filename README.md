# DQ魔法エフェクトカメラ

収録呪文: **マダンテ / メラゾーマ / ホイミ / ラリホーマ**（パネルのプルダウンか数字キー1〜4で切替）

ドラクエ11風の魔法エフェクトをWebカメラ映像にリアルタイム合成し、そのまま録画できるツール。
インストール不要、ブラウザ（Chrome / Edge 推奨）だけで動きます。

## 使い方

1. `index.html` をダブルクリックして開く（Chrome推奨）
2. 「▶ カメラを起動する」→ カメラ許可 → 初回は手認識モデルの読込に10秒ほど
3. **詠唱**: 片手（または両手）を頭上に上げる → 金色のルーン帯が回転しながら展開
4. **発動**: 「まりょくが みなぎった！」表示後、手を振り下ろす → 選択中の呪文が発動（3秒待つと自動発動）

| 呪文 | 演出 |
|---|---|
| 1: マダンテ | ピンク紫の大爆発＋光柱＋ダメージ数字×2 |
| 2: メラゾーマ | 火球が成長→炎の大爆発＋ダメージ数字 |
| 3: ホイミ | 降り注ぐ光と緑のきらめき＋緑の回復数字 |
| 4: ラリホーマ | ピンク紫の眠りの霧＋ただようＺ |
5. ダメージ数字とメッセージウィンドウが自動表示

※ もし真っ暗なまま・モデル読込に失敗する場合は `start-server.bat` を実行してから
表示されるURL（http://localhost:8123）で開いてください。

## 録画

- 「⏺ 録画開始」→ 演技 → 「⏹ 録画停止」で動画ファイル（mp4 or webm）が自動ダウンロードされます
- 効果音（合成SFX）も動画に入ります。不要なら「効果音」チェックを外す
- 左上のパネルは録画に映りません（☰ボタンで折りたたみ可）

## キー操作

| キー | 動作 |
|---|---|
| Space | 手動発動（詠唱→自動で発動。撮影係が押す用） |
| 1〜4 | 呪文切替（マダンテ/メラゾーマ/ホイミ/ラリホーマ） |
| D | ダメージ数字だけ出す |
| R | 録画開始/停止 |

## 設定

- **となえる人**: メッセージの名前（「ベロニカは まりょくを ためている！」）
- **ダメージ/回復**: ダメージ・回復数字の値（マダンテなら9999、ホイミなら35などに変えると雰囲気が出ます）

## 会議モード（マダンテで会議を強制終了）

Zoom会議中にマダンテを発動すると、爆発と同時に**「全員に対して会議を終了」**が実行されるモード。
自分がホストの会議でのみ成立します（ゲストだと退出になる）。**身内の会議・飲み会MTG専用**で。

### 仕組み

```
[Webカメラ] → [このページ(Chrome)] → [OBSウィンドウキャプチャ] → [OBS仮想カメラ] → Zoom
                    └─ マダンテ爆発ピークで localhost:8124 のヘルパーに通知
                                          └─ ヘルパーがZoomにAlt+Q→Enterを送信（全員に対して終了）
```

### セットアップ

1. **OBS Studio**（無料）をインストール
2. OBSで「ウィンドウキャプチャ」→ このページのChromeウィンドウを選択（ページはF11でフルスクリーン推奨）
3. OBSの「仮想カメラ開始」を押す
4. Zoomの設定 → ビデオ → カメラを「**OBS Virtual Camera**」に変更
5. `start-helper.bat` をダブルクリック（会議終了ヘルパーが常駐）
6. ページの「**会議モード**」にチェック →「せつぞくOK」と出れば準備完了
   （会議モード中は左上パネルがほぼ透明になり、OBSに映り込みません。マウスを乗せると復帰）

### 実行

会議中に片手を振り上げる → 全員にルーン詠唱が見える → 振り下ろす → マダンテ炸裂 →
ダメージ数字が出た瞬間、Zoomが「全員に対して会議を終了」で吹き飛びます。

### 注意

- 会議を終了するのは**マダンテだけ**。ホイミ等は演出のみ（変更は dq-helper.py の `KILL_SPELLS`）
- ターゲット変更も dq-helper.py 冒頭の設定で: `TARGET = 'zoom' / 'teams'(退出) / 'kill'(プロセス強制終了)`
- Zoomのウィンドウが最小化されていても復帰させてキーを送りますが、**本番前にテスト会議（1人Zoom）でリハーサル必須**。
  Alt+Q後の既定ボタンがホスト時は「全員に対して会議を終了」であることを確認しておくこと
- ヘルパーはローカル(127.0.0.1)のみ待ち受け。外部からは叩けません

## 撮影のコツ

- カメラから1.5〜2.5mくらいに立つ（手が画面内に十分な大きさで映る距離）
- **上げた手が画面内に収まる引きの構図**にしてください（頭上の手がフレームアウトすると認識が切れます）
- 明るい場所のほうが手の認識が安定します
- 手袋・長い袖は認識精度が落ちることがあります → その場合はSpaceキー発動で

## カスタマイズ（今後の拡張）

`index.html` の `<script>` 内がすべてのロジックです。
- 呪文を増やす: `SPELLS` オブジェクトにエントリを追加し、対応する `drawXxx()` を書く
  （マヒャド=水色の氷晶、ベギラゴン=黄色の稲妻、ザオリク=白金の光柱 など）
- ジェスチャー判定: `HANDS_UP_Y`（上げた判定の高さ）/ `RELEASE_Y`（振り下ろし判定）を調整
- 詠唱時間: `CHARGE_TIME`
